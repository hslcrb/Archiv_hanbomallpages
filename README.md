## 개요 / Overview

**한국어**

이 저장소는 경기도 수원 소재 **한봄고등학교** 공식 홈페이지(`hanbom-h.goesw.kr`)의 정적 아카이브입니다.

**English**

This repository is a static archive of the official website of **Hanbom High School** (`hanbom-h.goesw.kr`), located in Suwon, Gyeonggi-do, South Korea.
The initial archiving was performed on Sunday, March 1, 2026 (KST).

## 아카이빙 도구 / Archiving Tool

**한국어**

아카이빙에는 Google Chrome 확장 프로그램 **Save All Resources**(SingleFile 방식)를 사용하였습니다.
각 페이지는 모든 리소스(이미지, CSS, 폰트, 스크립트 등)가 포함된 단일 HTML 파일(`.html`)로 저장되었습니다.

**English**

The archiving was performed using the **Save All Resources** Google Chrome extension (SingleFile method).
Each page is saved as a single self-contained HTML file (`.html`) with all resources (images, CSS, fonts, scripts, etc.) embedded.

## 아카이브 범위 / Archive Scope

**한국어**

아카이빙된 항목은 다음과 같습니다:

- **학교소개** — 학교 소개 전 페이지(하위 포함)
- **입학안내** — 입학 안내 전 페이지(하위 포함)
- **결과 조회** — 결과 조회 전 페이지(하위 포함)
- **로그인 화면** — 경기도학교통합 로그인 페이지
- **마이페이지** — 마이페이지
- **통합검색** — 경기도수원교육지원청 통합검색
- **사이트맵** — 사이트맵
- **법적사항** — 개인정보처리방침, 영상정보처리기기 운영방침, 저작권보호정책, 이메일주소무단수집거부
- **방문통계** — 방문통계 보기
- **메인 페이지** — `한봄고등학교.html`

위 목록 이외의 페이지는 아카이브하지 않았습니다.

**English**

The following sections were archived:

- **School Introduction** — all pages under 학교소개 (including sub-pages)
- **Admissions Guide** — all pages under 입학안내 (including sub-pages)
- **Result Inquiry** — all pages under 결과 조회 (including sub-pages)
- **Login Page** — Gyeonggi School Integration login screen
- **My Page** — personal account page
- **Integrated Search** — Suwon Office of Education integrated search
- **Site Map** — site map page
- **Legal Notices** — Privacy Policy, CCTV Operation Policy, Copyright Protection Policy, Anti-Spam Email Collection Notice
- **Visit Statistics** — visitor statistics page
- **Main Page** — `한봄고등학교.html`

All other pages were **not** archived.
